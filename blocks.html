Content Blocks

Blocks are component-level styles for specific UI elements. They provide consistent styling for common content patterns and components.

Prose Block

The prose block provides optimized spacing for reading content. It sets a custom --flow-space for better reading experience.

Prose Content Example

This content uses the prose block, which sets a custom --flow-space for better reading experience. The prose block is designed to make text content more readable with appropriate spacing.

Notice how the spacing between paragraphs is optimized for reading, different from the regular flow composition. This creates a more comfortable reading experience, especially for longer content.

Features of Prose Block

  • Optimized spacing for reading
  • Consistent typography rhythm
  • Better readability for long-form content
  • Works well with headings and lists

"The prose block is perfect for articles, blog posts, documentation, and any content that needs to be easily readable."

You can use the prose block anywhere you have content that needs to be optimized for reading. It works particularly well for:

  • Article content
  • Blog posts
  • Documentation
  • Long-form text
  • User-generated content
